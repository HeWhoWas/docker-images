# k8s-tools

This image contains ``kubectl`` and several accompanying (and frequently used together) tools:

* ``awscli``
* ``kubesec``
* ``mysql`` and ``postgresql`` common packages
* ``jq``
* ``yq``
* ``helm``
* ``helmfile``
* ``promtool``

## versioning

The versioning for this image is based off the version of ``kubectl`` that has been packaged. This is the primary focus of this image, hence the version sticking to that particular component.
